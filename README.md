# Meru The Succubus - Steam Millennium Theme

A custom Steam theme featuring **Meru The Succubus**, built using the Millennium framework.

This is my very first time creating a custom Steam theme! I made this by following the official Millennium framework developer tutorial.


## 🛠️ How to Use
1. Download or clone this repository into your Millennium themes directory:
   `...\Steam\millennium\themes\Meru_Succubus_Theme`
2. Open Steam, go to the Millennium settings interface, and select **Meru_Succubus_Theme**.
3. Apply and reload Steam to enjoy the theme!
