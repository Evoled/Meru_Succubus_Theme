# TsukushiAria - Steam Millennium Theme

A custom Steam theme featuring **Tsukushi Aria**, built using the Millennium framework.

This is my very first time creating a custom Steam theme! I made this by following the official Millennium framework developer tutorial.
The theme centers around the VTuber **Tsukushi Aria (月紫アリア)**. Please check out and support her official X (Twitter) profile here: [@tsukushi_aria](https://x.com/tsukushi_aria).

## 🛠️ How to Use
1. Download or clone this repository into your Millennium themes directory:
   `...\Steam\millennium\themes\TsukushiAria`
2. Open Steam, go to the Millennium settings interface, and select **TsukushiAria**.
3. Apply and reload Steam to enjoy the theme!
